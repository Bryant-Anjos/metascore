# Metascore

Mobile app for tracking personal goals and habits, with a calendar view to mark daily completions.

## Features

- Create and manage goals/habits
- Calendar heatmap to track daily progress
- SQLite local storage (persists between sessions)
- Multilingual support (i18n via `expo-localization`)
- Dark mode support

## Tech Stack

- **React Native** + **Expo**
- **Expo Router** (file-based navigation)
- **expo-sqlite** for local persistence
- **NativeWind** (Tailwind CSS for React Native)
- **react-native-calendars**
- **i18n-js** + **expo-localization**
- **TypeScript**

## Getting Started

```bash
npm install
npx expo start
```
